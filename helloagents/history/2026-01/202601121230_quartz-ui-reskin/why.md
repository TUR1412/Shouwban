# Why: quartz-ui-reskin

## 背景
现有站点的业务逻辑、数据接口与运行时架构已经相对稳定，适合在“不破坏核心逻辑”的前提下，对视觉与交互进行更高标准的重塑。

## 目标
- 对标 Apple / Vercel 的顶尖审美：中性底色、精准层级、克制但有质感的高光与阴影。
- 全面引入微交互体系：hover/press/focus/loading/转场均有一致的动效语言。
- 保持 UI 与逻辑分离：仅改样式与动效层，不改核心数据与业务流程。

## 成功标准
- 视觉：关键容器（Header / 卡片 / 对话框 / Toast / 筛选条）统一玻璃、边框、阴影与字阶。
- 交互：关键 UI 反馈（如 Tabs 指示器）具备物理直觉的弹簧动效，并自动尊重减少动效偏好。
- 工程：`npm run verify` / `npm test` / `npm run build` 通过，且版本号统一 bump。
